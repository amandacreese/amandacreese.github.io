Key Documents Page - Quick Implementation Guide

About Key Documents Page

Key Documents Page allows for creation of a special page (or can be made global if you wish to do so) that allows for custom layout of the documents added through the lightbox.

Examples:

Example implementation can be viewed here:

Primary First Trust (GID Page Only)
Elmhurst GLT School (Global)

Implementation:

  1. Log in to your site and go into website admin > edit theme
  2. (optional) Create a GID page e.g
    content-page.page.page:06a4a351c46a4690a008244bc1a9ba21.html
  3. (optional) Paste in contents of content-page.page.html into newly created GID page.
  4. (optional) Add a class to body key-documents-page-special
  5. Create a additional.css file in your theme and paste in the CSS code from here
  6. If different icons are used SEE LIST WHICH SVG ICONS NEED TO BE PROVIDED BELOW, you will have to change them to data image using this tool . This will then have to be replaced for each icon (can be time consuming) Also ensure that you change colour in your SVG using fill attribute, alternatively you can CTRL + F within the stylesheet. Find the hex code used (original in CodePen is #50307c ) and replace it that way.
  7. In base.html include the newly added stylesheet before style.css :
    <link rel="stylesheet" type="text/css" href="{% theme_asset_file additional.css %}">
